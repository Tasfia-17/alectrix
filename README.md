# Alectrix
Your retro-themed study companion

Alectrix is a retro-themed study environment that brings personality and engagement to your learning routine through interactive tools and a cohesive 8-bit aesthetic.

A complete productivity platform featuring:

### Study & Productivity
- **Interactive Knowledge Map** - Visualize concepts and their relationships
- **Flashcard System** - Create and study with custom flashcards
- **Pixel Notetaker** - Clean, themed note-taking interface
- **Quest-Based To-Do List** - Manage tasks on a retro game map

### Wellness & Engagement
- **Mental Wellness Arcade** - Mini-games and breaks to prevent burnout
- **Mindful Meadow** - Guided breathing exercises with calming visuals
- **Gratitude Garden** - Journaling tool with virtual garden planting
- **Thought Validator** - CBT-based tool for mental wellness

### Personalization
- **Interactive Avatars** - Choose from pixel-art companions with dynamic dialogue
- **Customizable Environment** - Mood board and study music player
- **Cohesive 8-Bit Theme** - Consistent retro aesthetic throughout

## Tech Stack
- **Frontend**: React 19, TypeScript, Tailwind CSS
- **Architecture**: Build-less development using Import Maps
- **Styling**: Custom pixel-art design system
- **Deployment**: Vercel
- **State Management**: React Hooks with localStorage persistence

## Technical Implementation
### Key Features
- **Zero-Build Architecture** - Modern React/TypeScript app running directly in browser
- **Custom Design System** - Pixel-art UI components built with Tailwind CSS
- **Interactive SVG Components** - Animated avatars and visual elements
- **Client-Side State Management** - Efficient data handling with React hooks
- **Local Persistence** - Data saved across sessions using browser storage

### Architecture Highlights
- **Build-less Development** - No Webpack/Vite, using CDN-hosted dependencies
- **Component-Based Design** - Reusable, themed React components
- **Responsive Design** - Works across desktop and mobile devices
- **Performance Optimized** - Fast loading with minimal dependencies

## Running Locally
```bash
# Clone the repository
git clone https://github.com/your-username/alectrix.git
cd alectrix

# Serve with any static server
python -m http.server 8000
# or
npx http-server . -p 8000
```
Visit `http://localhost:8000` to view the application.
